# homebrew-whatport

Homebrew tap for [WhatPort](https://github.com/darrylmorley/whatport), the macOS menu bar app showing real-time USB-C port status (protocol, speed, power).

## Install

```sh
brew install --cask darrylmorley/whatport/whatport
```

Or tap first, install second:

```sh
brew tap darrylmorley/whatport
brew install --cask whatport
```

## Update

```sh
brew upgrade --cask whatport
```

## Uninstall

```sh
brew uninstall --cask whatport
```

Use `brew uninstall --zap --cask whatport` to also remove preferences and application support files.

## Requirements

macOS 14 (Sonoma) or later. Apple Silicon (M1+) only.
